レイヤー構成の説明 必須
各レイヤーの役割を表形式で説明
レイヤー 役割 主なクラス
Controller HTTPリクエストの受付、画面遷移制御、レスポンス返却 TaskController, UserController, AuthController
Service 業務ロジックの実装、トランザクション管理、ControllerとRepositoryの仲介 TaskService, UserService, AuthService
Repository（Mapper） データベースアクセス（CRUD処理）、SQL実行 TaskMapper, UserMapper
Entity データベーステーブルと対応するドメインオブジェクト Task, User
Form（DTO） 画面入力データの受け取り、バリデーション TaskForm, UserForm, LoginForm
View（Thymeleaf） 画面表示（HTML生成）、Modelデータの描画 list.html, detail.html, form.html, login.html
Security 認証・認可の管理、アクセス制御 SecurityConfig, UserDetailsServiceImpl

3 画面遷移フロー

全体の画面遷移図 必須
Webアプリケーション全体の画面遷移

+------------------------------+
| ログイン画面 |
| URL: /login |
+--------------+---------------+
　　　　　　　|
　　　　　　　| ログイン成功
　　　　　　　v
+------------------------------+
| タスク一覧画面 |
| URL: /tasks |
+---+--------------+-----------+
　　| |
　　| 詳細を見る | 新規作成
　　| /tasks/{id} | /tasks/new
　　v v
+------------------------------+　　　+------------------------------+
| タスク詳細画面 |　　　| タスク新規作成画面 |
| URL: /tasks/{id} |　　　| URL: /tasks/new |
+---+--------------+-----------+　　　+--------------+---------------+
　　| |　　　　　　　　　　　　　　　|
　　| 編集 | 一覧へ戻る　　　　　　　　　　| 確認へ
　　| /tasks/{id}/edit | /tasks　　　　　　　 | /tasks/confirm
　　v |　　　　　　　　　　　　　　　v
+------------------------------+　　　+------------------------------+
| タスク編集画面 |　　　| 確認画面（任意） |
| URL: /tasks/{id}/edit |　　　| URL: /tasks/confirm |
+---+--------------+-----------+　　　+------+-----------------------+
　　| |　　　　　　　　　　　　　　|
　　| 更新完了 | 詳細へ戻る　　　　　　　　| 登録実行
　　| /tasks/{id} | /tasks/{id}　　　　　 | POST /tasks/create
　　v |　　　　　　　　　　　　　　v
+------------------------------+　　　+------------------------------+
| タスク詳細画面 |　　　| タスク一覧画面 |
| URL: /tasks/{id} |　　　| URL: /tasks |
+---+--------------+-----------+　　　+------------------------------+
　　|
　　| 一覧へ戻る
　　| /tasks
　　v
+------------------------------+
| タスク一覧画面 |
| URL: /tasks |
+------------------------------+

【補足遷移】
タスク新規作成画面
├ 戻る → タスク一覧画面（/tasks）
├ 入力エラー → タスク新規作成画面（/tasks/new）
└ 確認画面なし → 登録後タスク一覧画面（/tasks）

確認画面（任意）
├ 戻る → タスク新規作成画面（/tasks/new）
└ 登録完了 → タスク一覧画面（/tasks）

タスク編集画面
├ 入力エラー → タスク編集画面（/tasks/{id}/edit）
├ 更新完了 → タスク詳細画面（/tasks/{id}）
└ 戻る → タスク詳細画面（/tasks/{id}）

タスク詳細画面
├ 編集 → タスク編集画面（/tasks/{id}/edit）
└ 一覧へ戻る → タスク一覧画面（/tasks）
