Scentory

作品概要 必須
どんなサービス・サイトか、ターゲットユーザーは誰か

作品名「Scentory」は、20代後半〜30代をターゲットにした香水ワークショップのWebサイトです。体験の魅力が伝わる写真や落ち着いたデザインを取り入れ、ワークショップ内容の紹介から予約までスムーズに進める導線を意識して設計しました。制作期間は約2か月です。

使用技術 必須

フロントエンド：HTML / CSS / JavaScript

バックエンド：Java / Spring Boot / MyBatis

データベース：MySQL

その他：Git / Figma

担当範囲 必須
企画・デザイン・フロントエンド・バックエンドなど

企画、デザイン、フロントエンド・バックエンド

工夫した点・アピールポイント 必須

工夫した点1：予約までの導線設計
ユーザーがスムーズに予約できるよう、サイト構成を工夫しました。トップページで体験内容が直感的に理解できるよう写真や説明を配置し、「How It Works」で体験の流れを説明したうえで予約ボタンへ自然につながる導線を設計しました。これにより、ユーザーが迷わず予約ページへ進めるように意識しました。

工夫した点2：デザインの統一感
香水ブランドのような上質感を表現するため、暗めのグレーを基調にボルドーをアクセントカラーとして使用しました。余白や写真のトーンも統一することで、落ち着いた雰囲気とブランドイメージを保つデザインを意識しました。

苦労した点：アクセスマップの表示
サイトに掲載するアクセスマップの画像生成がうまくいかず、デザインに合う形で表示する方法に悩みました。

どう解決したか
AIや検索を活用してマップ表示の方法を調査し、複数の方法を比較しました。その中でサイトのデザインに合う表示方法を選び、試行錯誤しながら実装しました。この経験から、問題が発生した際には情報収集と検証を繰り返しながら解決することの重要性を学びました。
