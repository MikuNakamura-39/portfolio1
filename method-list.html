1 Controller メソッド一覧

TaskController 必須
タスク管理コントローラーの全メソッド
メソッド名 HTTP URL 引数 戻り値 処理概要
list GET /tasks Model model String タスク一覧を取得してModelに追加し、list.htmlを返却
detail GET /tasks/{id} @PathVariable Long id, Model model String ID指定でタスクを取得し、Modelに追加してdetail.htmlを返却
showCreateForm GET /tasks/new Model model String 空のTaskFormをModelに追加し、form.html（新規作成画面）を表示
create POST /tasks @Valid @ModelAttribute TaskForm form, BindingResult result, RedirectAttributes redirectAttributes String バリデーション後、エラーがあればform.htmlを再表示、正常時は登録して一覧へリダイレクト
showEditForm GET /tasks/{id}/edit @PathVariable Long id, Model model String 既存データを取得してTaskFormに詰め、form.html（編集画面）を表示
update POST /tasks/{id} @PathVariable Long id, @Valid @ModelAttribute TaskForm form, BindingResult result, RedirectAttributes redirectAttributes String バリデーション後、エラーがあれば編集画面を再表示、正常時は更新して詳細または一覧へリダイレクト
delete POST /tasks/{id}/delete @PathVariable Long id, RedirectAttributes redirectAttributes String 指定IDのタスクを削除し、一覧画面へリダイレクト
search GET /tasks/search @RequestParam String keyword, Model model String キーワードでタスクを検索し、結果を一覧画面に表示
changeStatus POST /tasks/{id}/status @PathVariable Long id, @RequestParam String status, RedirectAttributes redirectAttributes String タスクのステータスを変更し、一覧または詳細画面へリダイレクト

2 Service メソッド一覧

TaskService 必須
タスクサービスの全メソッド
メソッド名 引数 戻り値 処理概要 呼び出すRepository
findAll なし List<Task> 全タスクを取得 taskMapper.selectAll()
findById Long id Task ID指定でタスクを取得 taskMapper.selectById(id)
findByUserId Long userId List<Task> ユーザー別タスク取得 taskMapper.selectByUserId(userId)
create TaskForm form, Long userId void FormからEntityに変換して登録 taskMapper.insert(task)
update Long id, TaskForm form void 既存タスクを更新 taskMapper.update(task)
delete Long id void タスクを削除 taskMapper.delete(id)
search String keyword List<Task> キーワードでタスク検索 taskMapper.search(keyword)
changeStatus Long id, String status void タスクのステータス変更 taskMapper.updateStatus(id, status)

3 Repository メソッド一覧

TaskMapper 必須
タスクマッパーの全メソッド
メソッド名 引数 戻り値 SQL種別 備考
selectAll なし List<Task> SELECT 全件取得。作成日時降順やID降順で並び替える想定
selectById Long id Task SELECT 主キーで1件取得。該当なしの場合はnull想定
selectByUserId Long userId List<Task> SELECT ユーザー単位で取得。作成日時降順や期限昇順で並び替える想定
insert Task task void INSERT @Insertで新規登録。title, content, due_date, priority, status, user_id, created_at, updated_atを保存
update Task task void UPDATE @Updateで主キー指定更新。updated_atも更新対象に含める想定
delete Long id void DELETE 主キーで削除。物理削除想定
search String keyword List<Task> SELECT タイトルや内容を対象にLIKE検索。部分一致検索（例: CONCAT('%', #{keyword}, '%')）想定
updateStatus Long id, String status void UPDATE ステータスのみ更新。updated_atも同時更新する想定
